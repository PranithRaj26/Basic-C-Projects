# First-Year C Projects

A collection of foundational C programs I wrote during my first year of B.Tech (ECE). I am using this repository to practice my core programming logic, work with user inputs, and get comfortable with version control.

## 📂 Included Programs

* **`Calculator.c`** - A basic command-line arithmetic calculator.
* **`CircleCalculator.c`** - Calculates the area and circumference of a circle based on user input radius.
* **`CompoundInterestProgram.c`** - Computes compound interest given principal, rate, and time.
* **`MadLibsGame.c`** - A fun text-based word replacement game using strings and standard input.
* **`MovieTicketProgram.c`** - Simulates a basic logic system for pricing or booking movie tickets.
* **`ShoppingCartProgram.c`** - A simple checkout system demonstrating arithmetic and variable management.
* **`WeightConversion.c`** - Converts weight between different units (e.g., kilograms and pounds).
